Terminal Dungeon Crawler

I made a Dungeon Crawler inspirsed game that you can play from your integrated terminal, nothing but a keyboard is needed. With cool graphics and an easy instruction set you can fend off enemies as they try to steal your throne. Survive as long as you can while trying to restore the crown in order to restore the kingdom!

Boat Game (In Progress)

I am currently working on a simulation game where you can explore a low-poly world and gather resources as well as build up civilizations from the ground up. It will offer multiple minigames in order to complete tasks with progressive gameplay that gets more involved the more you play.

Multithreaded DNS Resolver

This program I created allows you to input domains and resolve them to their respective DNS addresses. This program is multithreaded which allows for it to run at an impressive speed even when given tens of thousands of domains to resolve.
